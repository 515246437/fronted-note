外边距合并问题：

说明：原本这么写是为了让子元素的顶部距离父元素50排序

但是实际效果是子元素顶部紧贴父元素，margin-top的效果转移给了父元素

出现合并的条件如下，只要不满足其中之一就可以出现想要的效果（以上边距为例）：

父上边距和子上边距之间没有border（border: 5px solid red）

父上边距和子上边距之间没有padding（padding-top：1px））

父上边距和子上边距之间没有非空内容

父或子没有设置定位元素（除static和relative），overflow（除visible）和display：inline-block

父或资源都没有浮动
